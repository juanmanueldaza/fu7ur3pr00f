# fu7ur3pr00f Web UI — Implementation Plan

> **For Hermes:** Use subagent-driven-development skill to implement this plan task-by-task.

**Goal:** Add a FastAPI + Next.js web application to fu7ur3pr00f, exposing its career intelligence capabilities through a browser with l06_p0s3's "Vogue" design language.

**Architecture:** Monorepo with `/src/fu7ur3pr00f` (existing Python library), `/src/fu7ur3pr00f/api` (new FastAPI layer), and `/app` (new Next.js frontend). The API layer wraps existing gatherers, memory, and generators as REST endpoints + WebSocket for real-time updates. The frontend consumes the API via tRPC client and renders career data through the editorial design system.

**Tech Stack:**
- Backend: Python 3.13, FastAPI, uvicorn, Pydantic v2
- Frontend: Next.js 14+ (App Router), TypeScript strict, Shadcn/UI, TailwindCSS, Zustand, TanStack Query
- Typography: Playfair Display (headlines), Geist Sans (UI), Crimson accent (`#D00000`), Off-white canvas (`#FDFBF7`)
- Data: Existing ChromaDB stores, supabase/auth or local JWT for auth later

    req: GatherCVRequest,
    store: CareerKnowledgeStore = Depends(get_knowledge_store),

    return GatherResponse(
        source=DataSource.CV,
        sections_indexed=len(sections),
        chunks_created=chunks,

@router.post("/portfolio", response_model=GatherResponse)
async def gather_portfolio(
    req: GatherPortfolioRequest,
    store: CareerKnowledgeStore = Depends(get_knowledge_store),

```python
"""WebSocket handler for real-time career intelligence updates."""

import asyncio
import json
from typing import Any

from fastapi import WebSocket, WebSocketDisconnect


class ConnectionManager:
    """Manage active WebSocket connections."""

    def __init__(self):
        self.active: list[WebSocket] = []

    async def connect(self, ws: WebSocket):
        await ws.accept()
        self.active.append(ws)

    def disconnect(self, ws: WebSocket):
        if ws in self.active:
            self.active.remove(ws)

    async def broadcast(self, message: dict[str, Any]):
        for ws in self.active:
            try:
                await ws.send_json(message)
            except Exception:
                self.disconnect(ws)

    async def send_progress(self, ws: WebSocket, source: str, progress: float, data: dict | None = None):
        await ws.send_json({
            "type": "progress",
            "source": source,
            "progress": progress,
            "data": data or {},
        })


manager = ConnectionManager()


async def websocket_endpoint(ws: WebSocket):
    await manager.connect(ws)
    try:
        while True:
            # Keep connection alive, receive client commands
            data = await ws.receive_text()
            msg = json.loads(data)
            # Route: {"action": "subscribe", "source": "linkedin"}
            # Future: allow clients to subscribe to specific event types
    except WebSocketDisconnect:
        manager.disconnect(ws)
```

## Key Decisions Documented

1. **Monorepo**: `/app` alongside `/src` — same repo, distinct subproject
2. **Design language**: Shared with l06_p0s3 (Playfair + Geist + Crimson + no borders)
3. **API-first**: FastAPI wraps existing Python library, zero changes to gatherers/memory/generators
4. **No auth in Phase 1**: Single-user local app. Auth (Supabase) is Phase 5.
5. **No LangGraph in frontend**: The API calls existing scripts and functions directly. LangGraph is gone. If agent orchestration is needed later, it goes in the API layer as a specialist pattern.
6. **WebSocket for real-time**: Progress updates during gathering and analysis, not chat.
7. **tRPC consideration**: Using plain REST + fetch for Phase 1. tRPC client can be added in Phase 2 as a thin wrapper.
